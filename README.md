الموقع الشخصي لـ **حمزة** (HAMZA ROOT)، مطور ومتخصص في بناء الأنظمة الذكية، أتمتة سير العمل، وتطوير تطبيقات الويب والبوتات. يعرض الموقع الخبرات، المشاريع المميزة، والخدمات المقدمة.

![لقطة شاشة للموقع](screenshot.png) *(يُفضل إضافة لقطة شاشة للموقع هنا)*

---

## ✨ المميزات

- **تصميم عصري وجذاب** باستخدام خطوط `Syne` و `JetBrains Mono`.
- **مؤشر تفاعلي للماوس** (Custom Cursor) مع حلقة مضيئة.
- **خلفية متحركة** (Glowing Orbs) لإضافة عمق للصفحة.
- **قائمة تنقل متجاوبة** (Responsive Navbar) مع قائمة جانبية للأجهزة المحمولة.
- **شريط مهارات متحرك** (Ticker Tape) يعرض التقنيات المستخدمة.
- **أرقام إحصائية متحركة** (Animated Counters) لتوضيح الخبرة.
- **بطاقات مشاريع** مع تأثيرات حركية ومحاكاة لطرفية (Terminal).
- **نموذج تواصل** مع رسالة تأكيد ناجحة.
- **حركات ظهور سلسة** (Scroll Reveal Animations) عند التمرير.
- تصميم متجاوب بالكامل مع جميع أحجام الشاشات.

---

## 🛠️ التقنيات المستخدمة

- **HTML5**: هيكل الموقع.
- **CSS3**: التصميم، التنسيق، والحركات (Animations)، مع استخدام متغيرات (`variables`) لتنسيق الألوان.
- **JavaScript (ES6+)**: جميع التفاعلات والحركات الديناميكية مثل:
  - حركة مؤشر الماوس.
  - إظهار وإخفاء القائمة في الجوال.
  - شريط المهارات المتحرك.
  - عداد الأرقام الإحصائية.
  - إرسال النموذج وعرض رسالة النجاح.
  - تأثيرات الظهور عند التمرير.
- **Google Fonts**: خطوط `Syne` و `JetBrains Mono`.

## ✍️ التخصيص والتحسينات

يمكنك بسهولة تخصيص هذا الموقع ليتناسب مع هويتك:

- **المعلومات الشخصية**: غيّر النصوص في ملف `index.html` (الاسم، الوصف، المشاريع، الخدمات).
- **الروابط**: استبدل علامات `#` في روابط السوشيال ميديا داخل ملف `index.html` بروابط حسابك الحقيقية.
- **الألوان**: غيّر متغيرات الألوان في بداية ملف `style.css` (مثل `--primary`, `--background`).
- **إضافة صورة**: أضف لقطة شاشة للموقع (يفضل `screenshot.png`) في نفس المجلد.

### تحسينات مقترحة للمستقبل:

- ربط نموذج الاتصال بخدمة خارجية (مثل Formspree أو EmailJS) ليصبح قادرًا على إرسال الإيميلات فعليًا.
- استبدال روابط المشاريع (`#`) بروابط لمشاريع حقيقية على GitHub أو منصات العرض.
- إضافة المزيد من التفاصيل لكل مشروع في صفحة منفصلة.
